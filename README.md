- [plugin-oauth2](https://github.com/halo-sigs/plugin-oauth2) - Halo 2.0 的 OAuth2 第三方登录插件
- [plugin-moments](https://github.com/halo-sigs/plugin-moments) - Halo 2.0 的瞬间管理插件
