## 添加条目

1. Fork 本仓库
2. 在 `data.json` 中添加条目
3. 提交 PR

data.json 格式：

```json
{
  "type": "theme" | "plugin" | "other",
  "official": true | false,
  "repo": "https://github.com/your-username/your-repo",
  "name": "your-app-name",
  "description": "your-app-description",
  "appId": "your-app-id"
}
```
